## Deploying to Vercel

1. Go to [vercel.com](https://vercel.com) and sign in with GitHub
2. Click **New Project** → Import **expoxtechinc/dasbmsoe**
3. Set **Root Directory** to: `artifacts/dasbmse`
4. Set **Framework Preset** to: `Other`
5. **Build Command**: `npm install && npm run build`
6. **Output Directory**: `dist/public`
7. Click **Deploy** — done!

> ✅ The vercel.json file at artifacts/dasbmse/vercel.json handles routing automatically.
