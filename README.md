## Deploy to Vercel
1. Go to vercel.com → New Project → Import this repo
2. Root Directory: `artifacts/dasbmse`
3. Build Command: `pnpm install && pnpm run build`
4. Output Directory: `dist/public`
5. Click Deploy!
